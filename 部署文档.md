## 二、快速部署（一键脚本）

### 2.1 上传项目代码

```bash
# 在服务器创建目录
sudo mkdir -p /opt/changzheng

# 方式一：SCP上传（本地执行）
scp -r ./* root@服务器IP:/opt/changzheng/

# 方式二：Git克隆
git clone 你的仓库地址 /opt/changzheng
```

### 2.2 配置环境变量

```bash
sudo nano /etc/environment
```

添加以下内容：

```bash
MYSQL_ROOT_PASSWORD=changzheng123
WX_APPID=你的小程序AppID
WX_SECRET=你的小程序Secret
JWT_SECRET=changzheng-cloud-march-secret-key-2024-very-long
```

### 2.3 运行部署脚本

```bash
cd /opt/changzheng
chmod +x deploy.sh
sudo ./deploy.sh
```

脚本将自动安装并配置：
- JDK 17
- Maven
- MySQL 8.0
- Redis
- Node.js 18
- Nginx

## 三、手动部署

### 3.1 安装基础环境

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装依赖
sudo apt install -y curl wget git unzip software-properties-common

# 安装 JDK 17
sudo apt install -y openjdk-17-jdk
java -version

# 安装 Maven
sudo apt install -y maven
mvn -version

# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs
node -v
```

### 3.2 安装 MySQL 8.0

```bash
# 安装
sudo apt install -y mysql-server

# 启动并设置开机自启
sudo systemctl start mysql
sudo systemctl enable mysql

# 配置密码
sudo mysql
```

```sql
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'changzheng123';
CREATE DATABASE changzheng_db DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
FLUSH PRIVILEGES;
EXIT;
```

```bash
# 初始化数据库
mysql -u root -pchangzheng123 changzheng_db < /opt/changzheng/sql/V1__init_schema.sql
mysql -u root -pchangzheng123 changzheng_db < /opt/changzheng/sql/V2__init_data.sql
```

### 3.3 安装 Redis

```bash
sudo apt install -y redis-server
sudo systemctl start redis-server
sudo systemctl enable redis-server

# 验证
redis-cli ping  # 应返回 PONG
```

### 3.4 安装 Nginx

```bash
sudo apt install -y nginx
sudo systemctl start nginx
sudo systemctl enable nginx
```

## 五、配置 Nginx

```bash
sudo nano /etc/nginx/sites-available/changzheng
```

写入以下配置：

```nginx
server {
    listen 80;
    server_name localhost;

    # 管理后台前端
    location / {
        root /var/www/changzheng-admin;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # 管理后台 API
    location /api/admin/ {
        proxy_pass http://127.0.0.1:8085/admin/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        client_max_body_size 10m;
    }

    # 小程序 API
    location /api/ {
        proxy_pass http://127.0.0.1:8081/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}
```

启用配置：

```bash
sudo ln -sf /etc/nginx/sites-available/changzheng /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default
sudo nginx -t
sudo systemctl reload nginx
```

## 六、创建系统服务

### 6.1 认证服务

```bash
sudo nano /etc/systemd/system/changzheng-auth.service
```

```ini
[Unit]
Description=Changzheng Auth Service
After=network.target mysql.service redis.service

[Service]
Type=simple
User=root
Environment="MYSQL_HOST=localhost"
Environment="MYSQL_PASSWORD=changzheng123"
Environment="REDIS_HOST=localhost"
Environment="JWT_SECRET=changzheng-cloud-march-secret-key-2024-very-long"
Environment="WX_APPID=你的AppID"
Environment="WX_SECRET=你的Secret"
ExecStart=/usr/bin/java -jar -Xms256m -Xmx512m /opt/changzheng/changzheng-auth/target/changzheng-auth-1.0.0.jar
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

### 6.2 管理服务

```bash
sudo nano /etc/systemd/system/changzheng-admin.service
```

```ini
[Unit]
Description=Changzheng Admin Service
After=network.target mysql.service

[Service]
Type=simple
User=root
Environment="MYSQL_HOST=localhost"
Environment="MYSQL_PASSWORD=changzheng123"
ExecStart=/usr/bin/java -jar -Xms256m -Xmx512m /opt/changzheng/changzheng-admin/target/changzheng-admin-1.0.0.jar
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

### 6.3 启动服务

```bash
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start changzheng-auth
sudo systemctl start changzheng-admin

# 设置开机自启
sudo systemctl enable changzheng-auth
sudo systemctl enable changzheng-admin

# 查看状态
sudo systemctl status changzheng-auth
sudo systemctl status changzheng-admin
```

## 八、HTTPS 配置

### 8.1 申请证书（Let's Encrypt）

```bash
# 安装 certbot
sudo apt install -y certbot python3-certbot-nginx

# 申请证书（需要域名解析到服务器）
sudo certbot --nginx -d 你的域名

# 自动续期测试
sudo certbot renew --dry-run
```

### 8.2 微信小程序域名配置

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发管理」>「开发设置」>「服务器域名」
3. 添加 `request合法域名`：`https://你的域名`

## 十二、常见问题

### Q1: 服务启动失败？

```bash
# 检查依赖服务
sudo systemctl status mysql redis

# 检查端口占用
sudo netstat -tlnp | grep 808

# 查看详细错误
sudo journalctl -u changzheng-auth -n 100
```

### Q2: 小程序无法连接？

1. 确认服务器防火墙开放 80/443 端口
2. 小程序正式版必须使用 HTTPS
3. 检查域名是否在小程序后台配置

### Q3: 数据库连接失败？

```bash
# 检查 MySQL 状态
sudo systemctl status mysql

# 检查连接
mysql -u root -pchangzheng123 -e "SELECT 1"

# 检查权限
mysql -u root -pchangzheng123 -e "SHOW DATABASES"
```

### Q4: Nginx 502 错误？

1. 检查后端服务是否运行
2. 检查代理地址和端口是否正确
3. 查看 Nginx 错误日志

## 十三、安全建议

1. **修改默认密码**：MySQL、管理后台默认密码

3. **定期备份**：设置数据库自动备份
4. **更新系统**：定期更新系统补丁
5. **启用 HTTPS**：生产环境必须配置
